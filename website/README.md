# AirmonGUI Website

React/Vite website for AirmonGUI. This folder contains the public-facing project site:
hero, feature summaries, workflow explanation, download/GitHub call-to-action, and footer.

## Stack

- React 18
- TypeScript
- Vite 5
- Tailwind CSS
- shadcn/ui-style Radix components
- lucide-react

## Setup

```bash
cd website
npm install
```

## Running

```bash
npm run dev
```

Open `http://localhost:8080`.

## Scripts

| Command | Purpose |
|---|---|
| `npm run dev` | Start the website dev server |
| `npm run build` | Build production assets |
| `npm run build:dev` | Build with development mode |
| `npm run lint` | Run ESLint |
| `npm run preview` | Preview the production build |

## Content Guidelines

- Keep copy accurate to the local AirmonGUI app and backend.
- Link users to the repository and root README for installation details.
- Keep the security posture clear: authorized wireless testing only, local-first operation,
  and no telemetry claims unless the application remains telemetry-free.
- Avoid promising cloud features or remote operation that the app does not provide.

## Build

```bash
npm run lint
npm run build
```

The production build output is generated by Vite and should normally stay uncommitted.
